Sort by Cluster

Sorts the data according to the position in the clusters. This ensures that the order of the rows in the dendrogram is maintained in the table.

Detailed description: Sort by Cluster node

Dialog Options

ascending
Selects one of the possible orders.

Ports

Input Ports
0 A tree model from the New Hierarchical Clustering node.
1 Data with same keys (in the same order) as the Cluster tree has.
Output Ports
0 Sorted data
