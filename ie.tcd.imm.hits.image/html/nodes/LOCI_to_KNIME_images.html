LOCI to KNIME images

Based on the information from LOCI Plate Reader it converts the images present in that format to KNIME image format.

Dialog Options

combine channels
When selected the first (at most) 3 (grey-scale) channels will be combined to RGB images.

Ports

Input Ports
0 A port with references to images.
1 A port where the readers are present.
Output Ports
0 Table containing the images
