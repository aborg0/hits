Supported features

LOCI Plate Reader

It supports importing the known Bio-Formats image formats.

LOCI Plate Viewer

Supports HiLite, manual adjustment of colours.

Limitations

LOCI Plate Reader

It is not possible to select to import only a specified subfolder from input folder. The preview of images might be problematic in some cases.

LOCI Plate Viewer

Only 96 well plates are supported. Clicking on missing wells might cause problems.

Not supported features
