LOCI Plate Viewer node overview

This node is able to visualise the images read by LOCI Plate Reader node. This view gives a basic option to show images, but allows you to track the HiLites, and HiLite different wells. The node description shows the overview of the functionality, while this page gives a more detailed explanation.

Input data

First data port (image references)

This table should enumerate the images available from the readers. The format of the input is the same as the first outport's format of LOCI Plate Reader, maybe with other columns (order independent).

Expected format

There should be a column with name LOCI-Plate and textual type. Must not contain missing values (?).

The LOCI_Row column with textual type is also mandatory. The values should be from A to H. Must not contain missing values (?).

The LOCI_Column column has to be natural number (Integer) type, also mandatory, between 1 and 12.

The LOCI_Field column has to be natural number (Integer) type, also mandatory, values are from 1.

The LOCI_Z column has to be list of real number (Double) type, also mandatory.

The LOCI_T column has to be list of real number (Double) type, also mandatory.

The LOCI_C column has to be list of textual (String) type, also mandatory.

The LOCI_Id column has to be a textual (String) type, also mandatory.

The Image_Id column has to be a natural number (Integer) type (values from 0), also mandatory.

The LOCI Plate Reader node's first outport's result is compatible with this port.

Second data port (image join table)

This should contain the Bio-Formats readers

Expected format

  • LOCI-Plate text (String) type. The plate identifier goes here, mandatory.
  • LOCI-Row text (String) type. The row name (from A to H), might be empty, mandatory.
  • LOCI-Column natural number (Integer) type. The column position (from 1 to 12), might be empty, mandatory.
  • LOCI-Field natural number (Integer) type. The field position (from 1), might be empty, mandatory.
  • LOCI-Z list of real numbers (Double) type. The Z position, might be empty, mandatory.
  • LOCI-T list of real numbers (Double) type. The time position, might be empty, mandatory.
  • LOCI-C list of text (String) type. The channels, might be empty, mandatory.
  • LOCI LOCI image reader type. The actual image readers, should not be empty, mandatory.
  • OME-XML text (String) type. The metadata as xml, should not be empty, mandatory.
  • LOCI-Id text (String) type. The identifier of the reader, should not be empty, mandatory.

The order of colums is not important.

Controls

Mouse control

Zooming

Hold the Ctrl key, and scroll with your mouse wheel.

Scrolling

Possible with mouse wheel, or by holding the main mouse button, and moving the mouse.

Other controls

Selection

Selecting a value is possible by clicking on the button, or other form of control. With the zoom utility either changing the number or moving the slider have the same effect.

Adjust colour settings

By right clicking on the channel name you can change the colours assigned to that channel, or modify the channel.

To modify the range of a channel you have to find the two grey vertical lines on the histogram (might be necessary to zoom out from the popup menu), click on one of them, and click on a new position. This will change not just the preview of the image, but also the preview of the colours of that channel (without the possible inversion) above the histogram.

Zooming

Menu

HiLite

The common, usual HiLite menu. The smallest unit of HiLite is a well.

Colours

In this menu you can invert the colours used to visualise the image, or modify how the combine different channels.

Invert

Inverts the colours used, very simple operation, takes effect on all channels' colours.

Additive

The colours of the channels are added with other channel colours (for each R, G, B colour components independently) with truncated (0 - 255) values.

Minimum

The colours of the channels are combined with other channel colours (for each R, G, B colour components independently) by selecting the minimal values.

Maximum

The colours of the channels are combined with other channel colours (for each R, G, B colour components independently) by selecting the maximal values.
