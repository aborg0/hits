Merge

This node is a bit like an anti-sort. If you have a list something like this: 

Col1 |Col2
A    |1
A    |2
A    |3
B    |4
B    |5
B    |6
, and you want to merge the Col1 with itself, the result will look like this: 
Col1 |Col2
A    |1
B    |4
A    |2
B    |5
A    |3
B    |6
. This might be useful for the Pivot node.

Dialog Options

Independent columns
The columns not used in the reordering.
Columns used to reorder
The columns of reorder.
Sort in memory
Faster in-memory operation, only this is supported.
Reverse the order in a block
Reverses the order of the rows in the generated blocks.
Halt on error
If the reordering is not possible consistent (because there are missing rows from a pattern) it will stop, else just writes an error to the Console.
