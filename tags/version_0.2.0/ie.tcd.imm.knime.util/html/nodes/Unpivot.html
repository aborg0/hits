Unpivot

Converts a table -based on its column's names- to another table. The parts of the names will become values.
The column names can be changed by clicking on the preview table (only for the new ones).

Dialog Options

Pattern
A regular expression on column names. The non-grouping part is denoted by (?:something), the grouping parts will create the new columns. It shows the result structure.
HiLite Strategy
Support HiLite or not.

Ports

Input Ports
0 A table with pivoted columns.
Output Ports
0 The new table
