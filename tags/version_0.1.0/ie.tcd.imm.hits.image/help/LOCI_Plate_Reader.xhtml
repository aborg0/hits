LOCI Plate Reader node overview

This node is able to read images using the (slightly modified) Bio-Formats framework. The node description shows the overview of the functionality, while this page gives a more detailed explanation.

Result data

mapping to readers

These are the references to the readers in the second outport.

Structure

  • LOCI-Plate - The identifiers of the plate, textual type (S).
  • LOCI-Row - The row name, textual type (S), from A to H.
  • LOCI-Column - The column position, integral type (I), from 1 to 12.
  • LOCI-Field - The field identifier, integral type (I), from 1.
  • LOCI-Z - The position inside the well (Z dimension), list of decimal type (D) values.
  • LOCI-T - The time since the first image taken from that field (t dimension), list of decimal type (D) values.
  • LOCI-C - The channels of the image, list of textual type (S) values.
  • LOCI-Id - The identifier of the reader in the second outport, textual type (S).
  • Image-Id - The identifier of the actual image within the reader, textual type (S).

readers

These are the actual Bio-Formats readers.

Structure

  • LOCI-Plate - The identifiers of the plate, textual type (S).
  • LOCI-Row - The row name, textual type (S), from A to H, might be empty indicating that all of the rows on the plate are belonging to the reader in this row.
  • LOCI-Column - The column position, integral type (I), from 1 to 12, might be empty indicating that all of the columns on the plate are belonging to the reader in this row.
  • LOCI-Field - The field identifier, integral type (I), from 1, might be empty indicating that all of the fields on the well are belonging to the reader in this row.
  • LOCI-Z - The position inside the well (Z dimension), list of decimal type (D) values, might be empty indicating that all of the planes in the well are belonging to the reader in this row.
  • LOCI-T - The time since the first image taken from that field (t dimension), list of decimal type (D) values, might be empty indicating that all of the time points on the well are belonging to the reader in this row.
  • LOCI-C - The channels of the image, list of textual type (S) values, might be empty indicating that all of the channels on the plate are belonging to the reader in this row.
  • LOCI - The actual reader, LOCI reader type.
  • OME-XML - The textual representation of the metadata associated with the reader, textual type (S).
  • LOCI-Id - The identifier of the reader, textual type (S).

metadata
