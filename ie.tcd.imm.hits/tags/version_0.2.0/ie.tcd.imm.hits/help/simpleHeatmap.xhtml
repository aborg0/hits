Simple Heatmap Overview

This node shows the values of the columns coded as colours.

The node description shows the overview of the functionality, while this page gives a more detailed explanation.

View

Preferences in view

The view allows the user to change the colours for each column using the Colours tab. For details check Colour Settings section.

On General tab the width/height of a single rectangle can be adjusted. Also the position of the row ids, the shown statistics (related to the HiTS project: details) and the values at that position can be adjusted.

View controls

The selection and HiLiting works as in other views. The tooltips show the actual value, and the column/row information. (The row id is in parenthesis with bold font at the end.)

Saving the generated image

You can save the visible part of the view (File/Export as/), or the whole generated heatmap (File/Export as PNG or File/Export as SVG) without the controls.
