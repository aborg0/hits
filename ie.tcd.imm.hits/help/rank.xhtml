Rank node overview

This node allows you to rank your data by using different parameters/statistics. The node description shows the overview of the functionality, while this page gives a more detailed explanation.

Configuration

Well annotation

The well annotation field should contain a column (well annotation or final well annotation) with values like pos, sample, ...

Rank prefix

The prefix for the result rank columns.

Ranking groups

Here you can select defined grouping of the values to rank. Possible values:

  • experiment - each experiment (and parameter, normalisation) is creating a group independently from the replicate property.
  • plate - each plate (and experiment, parameter, normalisation) is creating a group independently from the replicate property.
  • replicate - each experiment, replicate group (and parameter, normalisation) is creating a group.
  • plateAndReplicate - each physical plate (and parameter, normalisation) is creating a group.

Statistics

Select the statistics you want to rank.

Parameters

The parameters to rank. (With the statistics.)

Ranking criteria

Consist of a number and a sign. The + sign means the normal ranking, the - sign means the inverse ranking. The number is where the sign of the ranking changes.

Tie handling

Possible values:

  • increase - on tie the ranking will increase with the number of tied values
  • continuous - on tie the ranking will get the next number after the tie.

Input data

The input data should be similar to the first or second output of cellHTS2 node. It may contain more columns which will be unaltered.

Expected structure

  • Experiment
  • Normalisation method
  • log transform
  • Normalisation kind
  • Variance adjustment
  • Scoring method
  • Summarise method
  • Plate

All other columns are optional, but at least one data column is expected, and also a well annotation (a column which contain sample as a value for some rows).

Output data

The original content followed by the rank columns.

Structure

Same as the input data but it contains additional columns with the rank values.

The rank columns contents are integral numbers, and integral numbers + .5 for the non-sample wells.
