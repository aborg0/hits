Simple Configurator for cellHTS2

Reads the cellHTS2 configuration files to simply set the type (i.e. sample, neg, pos, empty) of the wells and the description of the experiment. Details here.

Dialog Options

Plate configuration file
The configuration for the well types. For further information check the cellHTS2 manual looking for the Plateconf.txt file description.
Description file
The details of the experiment. For further information check the cellHTS2 manual looking for the Description.txt file description.

Ports

Output Ports
0 Plate configuration description for CellHTS2 node.
1 Experiment description for CellHTS2 node.
