biomaRt Annotator

Adds some annotations from the BioMart databases using the biomaRt R package. There must be a running RServe instance (on default port) on the machine running R. Details here.

Dialog Options

biomaRt database
The database to use
biomaRt dataset
The selected dataset from the database
attributes
The attributes to add to the results

Ports

Input Ports
0 Any kind of input table which has a geneID column
Output Ports
0 The same as the input table with the annotations from biomart.
