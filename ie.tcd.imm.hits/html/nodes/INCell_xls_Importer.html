INCell xls Importer

With this node you are able to read the well mean values from a IN Cell Analyzer 1000, and convert it to cellHTS2 readable format.

The experiments has the following structure: It has plates with some replicates for each plates. Each plate has 96, or 384 wells. The wells contain the experimental environment. From each well there can be taken some photographs on predefined positions (fields), which contains the cells.

Dialog Options

Folder of xls files
From this folder will be selected the XLS files.
XLS files
You can select the Excel files belonging to this experiment. They will be used in this (alphabetical) order as the first plate's first replicate, first plate's second replicate, ..., first plate's k replicate; second plate's first replicate, ..., second plate's k replicate; ...; n plate's first replicate, n plate's second replicate, ..., n plate's k replicate.
Well count
Number of wells on each plates.
Plate count
Number of plates in the experiment.
Replicate count
Number of replicates (1, if you have no replicates, 2 if every plate has a pair with the same configuration, ...)
Annotation file
The position of the annotation file. It should contain in the first row the header: Plate Well GeneID GeneSymbol. The other rows are the tab separated values for the plates and wells. Every plate-well pair have to be enumerated in the plate/well order.
