Plate Heatmap

Shows the heatmap of the plates. It has many options to customise of the view.

On the first tab of controls (Controls) you may influence the appearance of the wells.

The second tab (Info controls) allows you to set modify the tooltips, the description of the wells.

The third tab (Info) shows information about the last clicked well, below that you may see the input table. The latter can be filtered by the hilited values. (Hint: try right click on it.)

The forth tab (Settings) will allow you to save or load your preferred settings. This is currently not working.

The last tab (Colours) makes it possible to modify the automatically selected colours/ranges.

The sidebar allows you to select which heatmap you want to see. You can adjust the parameter values to your needs. You may change the position of the control (right click/Move to), or change its appearance (right click/Change appearance. You can also influence the division of the circles/rectangles (right click/Change type). (The primary for the circles is the division anti-clockwise, the secondary is the radial. The primary for the rectangles is splits the rectangle in the X axis, while the secondary splits by the Y axis.)

From the menu you can influence the HiLite.

There is an option to export the heatmaps in 

png
file format from the file menu. Select Export all... for this option.

You may turn on or off the colour legend in the Legend menu. Further details.
