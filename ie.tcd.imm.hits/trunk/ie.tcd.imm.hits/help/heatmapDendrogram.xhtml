Dendrogram with Heatmap

This node is simply a view of the output of the New Hierarchical Clustering node (from the KNIME Labs, distance matrix nodes). The node description shows the overview of the functionality, while this page gives a more detailed explanation.

View

Preferences in view

Default Settings

The Mouse Mode, Fit to size, Background Color, Use anti-aliasing options are the same as in the other controls, like Scatter Plot node has.

The zoom out, Flip Vertical options do not work currently. The Heatmap width is the width of a cell in the heatmap, while the Clusters is the number of the clusters to show on the view (those are separated by horizontal black lines). The Show Values options makes the numbers for each cell visible or hide them. The Flip Horizontal button changes the orientation of the dendrogram, but not the heatmap columns.

Colours

For details check Colour Settings section.

Saving the generated image

You can save the visible part of the view (File/Export as/), or the whole generated heatmap (File/Export as PNG or File/Export as SVG) without the controls.

Changing the visible heatmap columns

There are some options to see the data on the view:

  • Columns from second port (no indication of first)
  • Columns only from the first port (present in second too)
  • Columns in first in front (all of the columns from second port)
  • Columns in first at back (all of the columns from second port)
  • Columns from second port (selecting the columns present in first)

What is visible on the view?

The dendrogram and the heatmap are always on. The selection, and the HiLiting is just like in other nodes.

When all of the columns are visible from the second inport, and the indicators are selected the vertical lines around the columns are indicating that the clustering based on those columns.
