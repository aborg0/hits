Supported features

cellHTS2

You may add gene annotations, log transform, normalise, score and summarise the scores using the cellHTS2 node and save the HTML results.

On configuration gives help about the selected formulae.

Set which columns are interesting and in which order.

biomaRt annotator

You can add biomaRt annotations to your data (from one source at a time).

Ranking

Ranking by plates, by experiment or physical plates; multiple experiments or normalisation methods; tie handling.

Plate Heatmap

Allows you to visualise data from different experiments, different normalisation methods, and you can rearrange the controls and influence the structure of the shown data.

The HiLiting works. You can change the used colours.

Handles the associated colours as the background colour if available an unambiguous colour information to that visual well.

You can influence the generated tooltips.

Browse the HiLited rows.

Export the heatmaps as png files.

Simple Heatmap

Supports HiLiting, changing the colours, you can select which statistics you want to examine.

Heatmap with Dendrogram

Supports HiLiting, changing the colours, selecting the columns to view.

Limitations

cellHTS2

  • Screen log is not tested yet, it may be not working at all... If the screen log is empty it works well.
  • The view does not support clicking on links.

Heatmap

HiLiting

You cannot HiLite a part of a visual well, HiLite of a part of a visual well will generate HiLite on screen for the whole visual well. The HiLites are not saved, so next time (if you close your project) you have to HiLite everything you wanted.

Ranking

  • No proper regulation interface (you have option only to select a general natural point an direction, cannot specify to each parameter-statistics pair).

Not supported features

cellHTS2

  • call generation
  • Category analysis
  • Two way analysis
  • Summarise channels
  • Negative sign for scoring of a parameter
  • Other than 96 plate format

Heatmap

  • Does not support saving of actual arrangement/preferences.
  • Other than 96 plate format
